Detailed Description

template<typename T>
class Vector< T >

Dynamic array implementation similar to std::vector.

This class provides a dynamic array that can resize itself automatically when elements are added or removed. It supports various operations such as insertion, deletion, and random access.

Template Parameters
TThe type of elements stored in the vector.
